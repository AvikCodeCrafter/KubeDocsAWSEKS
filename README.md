## 🛠 Prerequisites  
Before using these guides, ensure you have:  
- An **AWS Account** with EKS permissions.  
- **kubectl** installed:  
  ```sh
  aws eks --region <your-region> update-kubeconfig --name <cluster-name>
  
AWS CLI configured
```
aws configure
```
